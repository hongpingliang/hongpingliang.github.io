  • Installtion for device

    Download RMBC.CAB on the device. Select "Save" and save it in the folder "\System\StartUp\" (if "StartUp" folder does not exist, create one). Open "\System\StartUp\" folder and click on "RMBC.CAB" to install.

    Previous Version:RMBC_Back.CAB
    After installation, click on "Start" => "Programs" => "RMBC" to run

  • Configuration for device

    Click on the appropriate file and click "File" => "Save As...". Save the file in the directory "\System\RMBC\" (if RMBC folder does not exist, create one). Change the file name to "setting.txt".

    Please note that directory is different to CAB file. After saved, restart the program for setting to take effect

    setting_receive_new.txt for receive a new pallet.

    setting_shipping.txt for shipping pallets

    setting_rework.txt for rework a pallet

    For detail information, open the file and check the comment line (beginning with "#")
